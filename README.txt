README

1. How to start the game:
	Simply load index.html in the browser of your choice, the game will start as soon as its loaded. (Tested on Mac with Chrome 40.0, Firefox 35.0.1 and Safari 8.0.3)

2. How to play the game:
	The game starts as soon as its loaded in your browser.
	To move the character you can use the arrow or the WASD keys. Both work at the same time, no need to change settings.

3. How to change the difficulty of the game:
	Below the game window is a dropdown located where you can change the difficulty. Simply try them out to which one fits best for you. On every change the enemies and the player get resetted.
